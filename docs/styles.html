Styles

This webpage is a demonstration and test of the default Drip stylesheet.

Typography

Most inline elements are not modified from the browser defaults.

The inline elements that are modified are:

  • <a href=""> — links are styled for better color contrast, and an icon is appended to external links.
  • <sub> — subscript, as in C8H10N4O2, is sized explicitly to define a standard for similarly sized elements, such as the external link icon.
  • <sup> — superscript, as in 4th of July, is sized explicitly to define a standard for similarly sized elements, such as the external link icon.

The inline elements that are not modified but may have visual differences from regular text are:

  • <abbr> — abbreviation (example: CSS)
  • <b> — bold text
  • <code> — computer code
  • <del> — deleted text
  • <dfn> — definition
  • <em> — emphasized text
  • <i> — italicized text
  • <ins> — inserted text
  • <kbd> — textual user input (example: Ctrl + F5)
  • <mark> — marked text
  • <q> — quoted text
  • <ruby> — annotations (example: 漢 (Kan) 字 (ji))
  • <s> — strikethrough
  • <strong> — important text
  • <u> — non-textual annotation
  • <var> — variable

Layout classes

  • .container — limits the width on horizontal flows or the height on vertical flows.
  • .container-full — does not limit the size but adds padding.
  • .row — arranges elements on the same line.

Formatting classes

  • .sr-only — hides an element from view but makes it readable by screen readers. (For example, this text.)
  • .text-muted — renders text in a muted color.
